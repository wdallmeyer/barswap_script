# barswap_script
AutoHotKey 2 script for convenient Barswapping in Elder Scrolls Online

This script allows you to map the often clunky and cumbersome weapon-swapping mechanic in ESO to a toggle button (Side Mouse Button 4).
You remain on Bar 2 as long as the mouse button is held down, and the game automatically switches back to Bar 1 when the button is released.
To prevent the logic from getting stuck on Bar 2, built-in timers ensure a switch back to Bar 1 after a short period.
For this to work, you must change the keybindings for Bar 1 and Bar 2 in ESO; the script uses Numpad / and Numpad * by default.
You can customize this to your preference by adjusting the variables within the script.

Another feature of the script is the display of the active bar in the top-left corner of the ESO window. For this to work, ESO must be running in fullscreen windowed mode. Alternatively, you can of course use add-ons for this purpose.
